# AI量化交易信号系统 - 前端

## 技术栈
- React 18
- TypeScript
- Ant Design
- ECharts
- React Router

## 开发环境设置

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm start
```

3. 构建生产版本
```bash
npm run build
```

## 项目结构
```
src/
├── components/     # 可复用组件
├── pages/         # 页面组件
├── assets/        # 静态资源
├── api/           # API 接口
└── utils/         # 工具函数
```

## 主要功能
- 资产类型选择
- 交易对选择
- 策略回测
- 报告展示
